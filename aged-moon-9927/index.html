Introduction to Streamlit

Build and share powerful data apps in minutes

What is Streamlit?

Streamlit is an open-source Python library that makes it easy to create beautiful, custom web apps for machine learning and data science.

It allows data scientists and engineers to turn data scripts into shareable web applications in minutes, all in pure Python, without needing front-end development experience.

Why Use Streamlit?

  • Simplicity: Build apps with simple Python scripts. No need for HTML, CSS, or JavaScript.
  • Speed: Go from concept to a deployed app in minutes, not days.
  • Interactive: Easily add widgets like sliders, text inputs, buttons, and checkboxes to make your apps dynamic.
  • Data-centric: Designed specifically for data scientists and ML engineers, with built-in support for data visualization, caching, and more.
  • Open Source: Free to use, with a vibrant community and active development.

How Does It Work?

You write Python scripts using Streamlit's intuitive API. Streamlit then handles the heavy lifting:

  • It sets up a local web server.
  • It generates the user interface based on your Python code.
  • It re-runs your script from top to bottom every time a user interacts with a widget (e.g., changes a slider value, clicks a button). This reactive model simplifies development significantly.
  • It offers powerful caching mechanisms to optimize performance for data-intensive operations.

A Simple Streamlit Example

Here's a basic Streamlit app that demonstrates a title, text, a slider, and a button.

import streamlit as st

st.set_page_config(layout="centered")

st.title("My First Streamlit App")

st.write("Hello, Streamlit! This is an introduction.")

# Add a slider widget
slider_value = st.slider("Select a number", 0, 100, 50)
st.write(f"You selected: {slider_value}")

# Add a button
if st.button("Say Hello"):
    st.success("Hello from the button!")

st.markdown("---")
st.write("This app demonstrates basic Streamlit widgets.")

Save this code as my_app.py (or any other .py file).

Running Your First Streamlit App

Follow these simple steps to run the example app:

  1. Install Streamlit: If you haven't already, open your terminal or command prompt and run:
    pip install streamlit
  2. Save the code: Save the example Python code from above into a file named my_app.py.
  3. Navigate to directory: Open your terminal or command prompt, and navigate to the directory where you saved my_app.py.
  4. Run from terminal: Execute the following command:
    streamlit run my_app.py

Your Streamlit app will automatically open in your default web browser (usually at http://localhost:8501).

What a Streamlit App Looks Like

Streamlit applications are designed with a clean, responsive, and modern user interface by default. Key features of the UI include:

  • Main Content Area: This is where your main app content (text, charts, tables, images, etc.) appears.
  • Sidebar (Optional): Many apps utilize a left-hand sidebar for input widgets (sliders, text inputs, selectors) to keep the main content clean.
  • Automatic Updates: As you interact with widgets, the app content updates in real-time without needing full page reloads.
  • The "Rerun" Button: In the top-right corner, you'll often see a "Rerun" button and menu for clearing cache or settings.

Data Visualization with Streamlit

Streamlit makes integrating powerful data visualizations incredibly simple. It supports popular plotting libraries like Matplotlib, Plotly, Altair, and also provides its own native charting methods for quick visualizations.

Here's an example of creating a dynamic line chart directly within your Streamlit app using `st.line_chart`:

import streamlit as st
import pandas as pd
import numpy as np

st.subheader("Interactive Line Chart Example")

# Create some dummy data for the chart
chart_data = pd.DataFrame(
    np.random.randn(20, 3), # 20 rows, 3 columns of random numbers
    columns=['a', 'b', 'c'] # Column names
)

# Display the line chart
st.line_chart(chart_data)

st.write("This chart is interactive: you can pan, zoom, and hover over data points.")
st.write("Streamlit intelligently renders charts from common data structures like Pandas DataFrames.")

This snippet demonstrates how easily you can turn numerical data into an interactive chart, a cornerstone of any data application.
