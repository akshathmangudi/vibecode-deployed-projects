About This Page

This is a very basic example of a self-contained web application. All its HTML, CSS, and JavaScript are bundled into a single `.html` file.

It's designed to be simple, functional, and demonstrate how a web application can exist without relying on external files once downloaded.

You can save this page to your computer and open it with any web browser, and it will work just like it does online.
